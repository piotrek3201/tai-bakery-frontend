## Description 📔
The web application of the confectionery store "Słodzianki" allows users to view the store's offer, add to cart and purchase products, as well as create their own cake. Users can create an account, so they can view previous orders and cancel orders. The app also includes an admin panel, where users can manage orders and products available in the store.
